# 🌱 AGRONODO

Agronodo es una solución tecnológica basada en IoT para el monitoreo agrícola en tiempo real, que permite a pequeños y medianos productores optimizar el uso del agua y mejorar la productividad de sus cultivos.

## 🚀 Problema
Muchos agricultores no cuentan con herramientas para medir las condiciones del suelo, lo que genera riego ineficiente, desperdicio de agua y bajos rendimientos.

## 💡 Solución
Agronodo integra sensores de humedad de suelo, temperatura y conectividad IoT (ESP32 + LoRaWAN) para capturar datos en campo y visualizarlos en una plataforma digital accesible.

## ⚙️ Arquitectura
Sensor → ESP32 → LoRaWAN → Plataforma IoT → Usuario
